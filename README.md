# Node 200 EJS Portfolio

This is my portfolio page built using the EJS template engine.

## Getting Started

- "npm run build"

- "npm start"

- Open on localhost:8080

### Installing

Run "npm install" in Terminal

## Running the tests

Run "npm test"

## Built With

- Node

- Express

- Bootstrap

- EJS
